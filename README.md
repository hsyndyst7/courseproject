# 🎓 Course Project

Bu proje, nesne yönelimli programlama (OOP) prensipleri kullanılarak geliştirilmiş bir **kurs / sınıf yönetim sistemi** uygulamasıdır. Proje, öğrenci, öğretmen ve sınıf ilişkilerini modelleyerek temel bir eğitim sistemi simülasyonu sunar.

## 🚀 Özellikler

- 📚 Sınıf (Classroom) oluşturma ve yönetme
- 👨‍🎓 Öğrenci ekleme / çıkarma
- 👩‍🏫 Öğretmen atama
- 📋 Listeleme ve bilgi görüntüleme
- 🔄 Nesneler arası ilişkilerin yönetimi (OOP)

## 🏗️ Kullanılan Teknolojiler

- C# (.NET)
- Nesne Yönelimli Programlama (OOP)
- Temel veri yapıları (List, Class vb.)
